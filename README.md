# 🌸🛵 Flowery Delivery [![codecov](https://codecov.io/github/YasmeenGad/flowery-delivery/graph/badge.svg?token=GVVWT7Q6N6)](https://codecov.io/github/YasmeenGad/flowery-delivery)

Flowery Delivery is a 🌷 Flutter-based floral delivery app that lets users explore, customize, and send flowers anywhere — quickly and beautifully.

---

## ✨ Features

- 🔐 **Secure Login & Signup**  
- 🎁 **Customize Your Gift**  
- 📍 **Live Order Tracking with Maps**  
- 📣 **Real-time Push Notifications**  
- 🤖 **Driver Auto-Registration via AI OCR** (License / ID Recognition using Google ML Kit)  
- 🗺️ **Google Maps SDK Integration**  
- 🔑 **Password Recovery**  
- 🧑‍💼 **Profile Management** (Photo + Data Update)

## 🛠️ Tech Stack

- 💙 **Flutter** / **Dart**: Cross-platform mobile app development  
- 🔥 **Firebase**: Firestore + Auth for backend and real-time sync  
- 🌐 **REST APIs**: Server communication  
- 🗺️ **Google Maps SDK**: Real-time location tracking  
- 🧠 **Google ML Kit**: OCR for auto-filling driver data  
- 🌀 **State management**:  **BLoC** for Main state management throughout the app  & **Provider** for Specifically used for real-time tracking states
- ⚙️ **CI/CD automation**:   GitHub Actions
- 📱 **Responsive UI**: Works across all screen sizes
